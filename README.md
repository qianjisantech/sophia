# Sophia - 在线协作文档平台

一个基于 Vue 3 + TypeScript + TDesign + Vite 构建的在线文档编辑器。

## ✨ 特性

- 📝 富文本编辑器：基于 Tiptap，支持标题、列表、引用、代码等丰富格式
- 📁 文档管理：文件夹分类、搜索、创建、删除等完整功能
- 💾 自动保存：编辑内容自动保存，无需手动操作
- 🎨 现代化 UI：基于 TDesign 组件库，界面简洁美观
- 📱 响应式设计：适配不同屏幕尺寸
- ⚡️ 快速开发：使用 Vite 构建工具，开发体验流畅

## 🛠️ 技术栈

- **框架**：Vue 3 (Composition API)
- **语言**：TypeScript
- **构建工具**：Vite
- **UI 组件库**：TDesign Vue Next
- **富文本编辑器**：Tiptap
- **状态管理**：Pinia
- **路由**：Vue Router

## 📦 快速开始

\`\`\`bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
\`\`\`

访问 http://localhost:5173 查看应用
